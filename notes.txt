# MVP Features
##Required Features
Add new vehicles  to the inventory by entering basic details such as make, model, year, mileage, and price.(DONE)
Enable potential customers to make to order for limited edition as well as run out time for thie production.
Customers should be able to see new models in stock as well as in dealerships
Customers should be able to update vehicle details such as pricing, availability status (e.g., available, sold, pending sale), and any other relevant information.
Customers should be able to view the current status of the inventory, including the number of available vehicles, sold vehicles, and vehicles pending sale.
